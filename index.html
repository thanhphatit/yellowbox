Automate Your Infrastructure

The official distribution center for Yellowbox CI/CD tools, Git utilities, and production-ready Helm Charts. Built for speed, scale, and multi-platform support.

Install CLI Tools

Install cicd-manager and git-push utilities instantly via our installation script (Linux & macOS).

Terminal curl -sSL https://<your-username>.github.io/yellowbox/scripts/install.sh | bash
Windows users? Browse /bin directory directly.

Helm Repository

Add the Yellowbox repository to your Helm client to deploy infrastructure charts like oblivion-sentinel.

Terminal helm repo add yellowbox https://<your-username>.github.io/yellowbox/charts/

helm repo update
